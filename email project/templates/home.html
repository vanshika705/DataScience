Project Details

In this project python,various Machine Learing tools such as data gathering, data filtering, data processing, CountVectorizer, TfidfVectorizer, Confusion matrix, Classification report are used. Along with this, various Machine Learing algorithms like BernoulliNB, GaussianNB, MultinomialNB, etc are used. It also includes some Python libraries such as pandas, numpy, seaborn, matplotlib, joblib, os, re, etc. When user enters the email , this project will predict that the given email is a spam or not.
